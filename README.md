# Instalación de Laravel con Docker

Paquetes y librerias necesarias para instalar laravel con docker


### Pre-requisitos 📋

```
-Docker
-Docker compose
-Laravel

```

### Instalación 🔧

```
    Ingresar al proyecto
    Ejecutar: 
            docker-compose build    
            docker-compose up -d
    Descargar el proyecto de laravel en la carpeta src

```
